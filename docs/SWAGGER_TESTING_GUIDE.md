# Swagger Documentation & Testing Guide

This guide explains how to use the interactive Swagger documentation to test the Umurava AI platform's backend APIs.

## 🔗 Access Links

- **Live Production API**: [https://umurava-backend-fdkp.onrender.com/doc](https://umurava-backend-fdkp.onrender.com/doc)
- **Local Development API**: [http://localhost:5000/doc](http://localhost:5000/doc)

---

## 🔑 Authentication Flow

Most endpoints are protected and require a Bearer Token.

1. **Login**:
   - Locate the **Auth** section in Swagger.
   - Expand `POST /api/auth/login`.
   - Click **Try it out**.
   - Enter your credentials (e.g., `aline.uwimana@umurava.africa` / `Recruiter2026!`).
   - Click **Execute**.
   - Copy the `token` from the response.

2. **Authorize**:
   - Scroll to the top of the page.
   - Click the green **Authorize** button.
   - In the text box, type `Bearer ` followed by your token (e.g., `Bearer eyJhbGciOiJIUzI1NiI...`).
   - Click **Authorize** and then **Close**.

## 🧪 Testing the Recruitment Lifecycle

### 1. Trigger AI Screening
- **Endpoint**: `POST /api/screening/{jobId}`
- **Description**: This starts the AI evaluation of all applicants for a specific job.
- **Status Change**: Observe that applications for this job will transition to `Under Review` and then `Screened`.

### 2. Verify Application Status
- **Endpoint**: `GET /api/applications`
- **Description**: Fetch all applications to verify the `status` field has been updated correctly.

### 3. Manual Status Management
- **Endpoint**: `PUT /api/applications/{id}`
- **Body**: `{ "status": "Shortlisted" }` or `"Interview"`, `"Hired"`, `"Rejected"`.
- **Goal**: Verify that manual overrides work and correctly reflect in the Applicant Portal timeline.

---

## 🛡️ Pro Tips for Testing
- **Clear Data**: Use the `DELETE /api/applications/{id}` endpoint if you need to reset a test application for a specific candidate.
- **AI Breakdown**: Check the `GET /api/screening/{jobId}` response to see the granular `skillBreakdown` and `aiReasoning` generated by Gemini.
